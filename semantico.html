HTML SEMÂNTICO


O HTML semântico é a forma de deixar o site com suas informações bem explicadas e compreensíveis para o computador, ajudando até mesmo em sua busca no Google e facilitando o entendimento de leitores de acessibilidade. Com o HTML semântico, ficou muito mais fácil de interpretar páginas.

Semântica é o estudo do significado e interpretação de palavras. Aplicando isso ao HTML, significa a melhor compreensão do HTML para o computador. Antes do HTML5, utilizava-se somente divs para dividir conteúdos e classes para estilizar, sem contar com a compreensão do próprio computador para “entender” o HTML.


Essas são as tags semânticas:

  • aside
  • article
  • footer
  • nav
  • main
  • header
